# 🐉 Dragon Slayer RPG

A browser-based text RPG built with **HTML, CSS, and JavaScript** where players explore a town, purchase weapons, battle monsters, and ultimately face the mighty **Tyrant the Dragon**.

This project was created to practice JavaScript fundamentals such as DOM manipulation, state management, arrays, objects, functions, and event handling.

---

## 🎮 Features

- ⚔️ Fight different monsters with unique stats
- 🏪 Visit the store to buy health and stronger weapons
- 🎒 Inventory system
- ⭐ Gain XP after defeating monsters
- 💰 Earn gold from battles
- 🛡️ Upgrade weapons to increase attack power
- 🐉 Defeat the final boss to win the game
- 🔄 Restart system after winning or losing
- 📱 Responsive modern UI

## ▶️ Running the Project

1. Clone the repository

```bash
git clone https://github.com/Sandy-YP-Holley/Dragon-Slayer.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or dependencies required.
